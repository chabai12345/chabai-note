# 🎮 侦探看板 · Detective Board

## 这是什么？

一个开源的剧情推理可视化工具——你在玩剧情堆成山的游戏（极乐迪斯科、逆转裁判、赛博朋克2077、P5R、星穹铁道……）时，再也不用靠脑子记几百号人谁是谁、谁杀了谁、谁爱谁了。

**卡片 + 连线 + 存档 = 你自己的推演墙。**

---

## 详细功能

### 📇 卡片系统

三种卡片类型：**人物**、**物品**、**地点**，每种用不同颜色区分。

每张卡片可以：
- 写标题 + 详细备注
- 上传本地图片（支持直接粘贴截图）
- 随意拖拽排版
- 双击/单击打开详情编辑框

卡片会自动分配 z-index，拖拽后置顶，操作手感自然。

### 🔗 智能连线

**每条连线独立控制：**
- 箭头样式：无箭头（普通连线）、单箭头（指向性关系）、双箭头（双向关系）
- 连线颜色：7 种颜色可随时切换
- 连线名称：给关系命名，比如「认识」「背叛」「同盟」
- 字号大小：10~30px 自由调整

**操作方式：**
1. 点击卡片顶部的图钉按钮，直接进入连线模式（默认无箭头）
2. 或者点击侧边栏「🔗 连线」按钮 → 弹出箭头样式选择 → 选好后依次点两张卡片
3. 连线生成后，**可以直接拖拽控制点来弯曲曲线**，避免连线遮挡卡片
4. 点击已有连线 → 浮窗弹出，修改名称、箭头、颜色、字号

**连线标注：** 名称标签会跟随贝塞尔曲线中点，即使拖拽弯曲也会自动调整位置。

### ✂️ 切断模式

点击连线浮窗中的「切断」选项，鼠标变成剪刀 ✂️，点击任意连线直接断开。
对比传统双击删除，剪刀模式更适合快速整理大量连线的场景。

### 🎮 存档系统

**多推演墙管理：**
- 点击侧边栏「🎮 档案」→ 弹出存档列表
- 支持**新建**推演墙、**重命名**、**读取**、**删除**
- 切换推演墙时**自动保存**当前进度，不会丢内容
- 首次使用自动创建「档案」存档

**保存机制：**
- 手动点击「💾 保存」随时存盘
- 切换存档时自动保存
- JSON 格式存储，路径 `data/`. 文件，可直接备份迁移

### 🎨 画布体验

- **软木板背景**，带暗纹点阵，像真的在墙上钉线索卡
- 画布外侧淡黄色边框，区分工作区域边界
- **滚轮缩放**、**拖拽平移**（类似 Figma / Miro 手感）
- 默认画布 8000×8000，远超实际使用需求，不担心出界
- SVG 渲染，连线使用贝塞尔曲线（Quadratic Bezier），边缘带阴影

### 📐 技术特点

- **纯前端 SPA**（单页应用），无框架依赖
- 后端仅提供 REST API（FastAPI）+ 静态文件服务
- 三键侧边栏：🎮档案 / 📇卡片 / 🔗连线，各有专属配色
- 所有数据保存在本地，不上传任何服务器

## 快速上手

```bash
pip install -r requirements.txt
python main.py
```

浏览器打开 `http://localhost:8080`。

### 操作三步走

1. **建卡**：侧边栏「📇 卡片」→ 选类型（人物 / 物品 / 地点）→ 卡片出现在画布
2. **连线**：点卡片顶部图钉 → 点另一张卡片 → 连线生成
3. **整理**：拖拽卡片排版，拖拽连线弯曲，点击编辑名称
